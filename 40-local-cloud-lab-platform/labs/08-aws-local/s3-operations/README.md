# AWS S3 Operations with LocalStack

**Difficulty**: intermediate | **Time**: 45 minutes | **Category**: aws-local

## Overview

Practice AWS S3 operations using LocalStack — a fully functional local AWS emulator. You will create buckets, upload and download objects, set bucket policies, enable versioning, create lifecycle rules, and generate presigned URLs. All commands run against LocalStack on localhost:4566 — no real AWS account needed.


## Getting Started

```bash
make run-lab LAB=08-aws-local/s3-operations
```
