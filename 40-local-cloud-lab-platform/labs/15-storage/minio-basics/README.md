# MinIO: S3-Compatible Object Storage Operations

**Difficulty**: beginner | **Time**: 45 minutes | **Category**: storage

## Overview

Practice object storage operations using MinIO — an S3-compatible storage server that runs locally. Learn bucket management, object operations, access policies, lifecycle rules, and the S3-compatible API. MinIO is already running as part of the core platform profile.


## Getting Started

```bash
make run-lab LAB=15-storage/minio-basics
```
