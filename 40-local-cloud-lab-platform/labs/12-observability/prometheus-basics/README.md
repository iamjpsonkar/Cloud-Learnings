# Prometheus: Metrics Collection and PromQL Queries

**Difficulty**: beginner | **Time**: 45 minutes | **Category**: observability

## Overview

Learn Prometheus fundamentals by exploring metrics from the running lab platform, writing PromQL queries, creating recording rules, and understanding metric types (Counter, Gauge, Histogram, Summary). All services are already instrumented.


## Getting Started

```bash
make run-lab LAB=12-observability/prometheus-basics
```

## Instructions

See `lab.yaml` for full task list, prerequisites, and validation steps.
