# Platform Orientation: Get to Know the Lab Environment

**Difficulty**: beginner | **Time**: 15 minutes | **Category**: foundations

## Overview

In this orientation lab, you will explore the local cloud lab platform itself. You will learn how to start services, run the lab runner, check the dashboard, and understand the Docker Compose profile system. No cloud account required.


## Getting Started

```bash
make run-lab LAB=00-foundations/platform-orientation
```

See `lab.yaml` for full task list, prerequisites, and validation steps.
