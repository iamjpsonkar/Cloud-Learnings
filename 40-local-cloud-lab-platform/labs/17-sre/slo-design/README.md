# SRE: Design and Implement SLOs and Error Budgets

**Difficulty**: intermediate | **Time**: 45 minutes | **Category**: sre

## Overview

Learn Service Level Objectives (SLOs) by defining them for the lab platform's own services. You will write SLO definitions, create Prometheus recording rules to compute SLIs, build an error budget dashboard in Grafana, and implement an alert that fires when the error budget is consumed.


## Getting Started

```bash
make run-lab LAB=17-sre/slo-design
```

See `lab.yaml` for full task list, prerequisites, and validation steps.
