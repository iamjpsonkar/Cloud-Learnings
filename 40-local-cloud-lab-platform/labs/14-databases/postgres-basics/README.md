# PostgreSQL Basics: SQL Operations and Administration

**Difficulty**: beginner | **Time**: 45 minutes | **Category**: databases

## Overview

Practice PostgreSQL fundamentals including DDL, DML, transactions, indexes, and basic administration. Connect to the local PostgreSQL instance running in Docker and perform real SQL operations.


## Getting Started

```bash
make run-lab LAB=14-databases/postgres-basics
```

## Instructions

See `lab.yaml` for full task list, prerequisites, and validation steps.
