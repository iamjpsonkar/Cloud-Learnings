# HashiCorp Vault: Secrets Management Fundamentals

**Difficulty**: beginner | **Time**: 45 minutes | **Category**: security

## Overview

Learn to use HashiCorp Vault for secrets management. You will write and read secrets, understand the KV secret engine, create policies, generate dynamic tokens, and learn why Vault is better than plain environment variables or config files.


## Getting Started

```bash
make run-lab LAB=11-security/vault-basics
```

See `lab.yaml` for full task list, prerequisites, and validation steps.
