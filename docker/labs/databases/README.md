# Lab: Databases

Practice with PostgreSQL, MySQL, MongoDB, and Redis — CRUD operations, indexes, backup, and restore.

## Objectives

1. PostgreSQL: CRUD, explain plans, indexes, transactions
2. MySQL: CRUD, user management, slow query log
3. MongoDB: CRUD, aggregation pipeline, indexes
4. Redis: data structures, expiry, pub/sub
5. Backup and restore all databases
6. Connection troubleshooting

## Prerequisites

```bash
./run.sh start data
```
