# Sandbox

Your free workspace. No instructions, no requirements — just experiment.

These directories are in `.gitignore` so your work won't be committed.

## docker/

Experiment with Dockerfiles, docker run commands, multi-stage builds, Docker Compose files.

## terraform/

Write Terraform configurations against LocalStack. Try modules, workspaces, backends.

## kubernetes/

Write Kubernetes manifests. Apply them to your local kind/k3d cluster.

## ansible/

Write Ansible playbooks targeting Docker containers.

## experiments/

Anything else — scripts, configs, notes.

---

**Tip**: Use `./run.sh start <profile>` to ensure the required services are running before experimenting.
