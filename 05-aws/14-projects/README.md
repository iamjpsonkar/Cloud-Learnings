# AWS Hands-On Projects

Practical projects that combine multiple AWS services into complete, deployable architectures. Each project builds on concepts covered in earlier sections.

---

## Projects

| File | Services | Difficulty |
|------|----------|-----------|
| [static-website.md](./static-website.md) | S3, CloudFront, ACM, Route 53 | Beginner |
| [secure-vpc.md](./secure-vpc.md) | VPC, Subnets, SGs, NACLs, NAT, VPC Endpoints | Intermediate |
| [3-tier-architecture.md](./3-tier-architecture.md) | EC2/ECS, ALB, RDS, ElastiCache, Secrets Manager, CloudWatch | Intermediate–Advanced |

---

## Suggested Order

1. **Static website** — Learn S3 hosting, CloudFront distribution, ACM certificate, and Route 53 alias records
2. **Secure VPC** — Build the network foundation all other projects rely on
3. **3-tier architecture** — Deploy a realistic production application pattern
