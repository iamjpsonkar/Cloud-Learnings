# Roadmaps

Learning roadmaps for three cloud engineering career paths. Each roadmap is a sequence of skills — build the foundation before the specialization, and the specialization before the advanced topics.

---

## Choose Your Path

```
Are you starting from scratch?
  └── Start with: Cloud Engineer Roadmap (foundation for all paths)

Do you want to focus on automation, pipelines, and delivery?
  └── After foundations: DevOps Engineer Roadmap

Do you want to focus on reliability, production systems, and incidents?
  └── After foundations: SRE Roadmap
```

## Roadmaps

| File | Path |
|------|------|
