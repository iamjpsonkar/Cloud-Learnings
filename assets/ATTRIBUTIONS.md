## Guidelines for Adding Attributions

When you add an image from an external source, add a row to the table above with:

- **File**: path relative to `assets/` (e.g., `images/aws/vpc-diagram.png`)
- **Source**: URL or description of where it came from
- **License**: e.g., CC BY 4.0, MIT, Public Domain
- **Notes**: any restrictions or attribution requirements

If an image has no clear license, do not include it in the repository.
